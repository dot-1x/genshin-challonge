# Genshin Challonge

A **Genshin Impact** tournament bracket simulation and draft pick tool for Spiral Abyss and Stygian Onslaught. Create single or double elimination brackets, simulate match results, and manage draft picks.

## Features

- **Bracket Simulation** — Single and double elimination tournament brackets for Spiral Abyss / Stygian Onslaught
- **Draft Pick System** — Simulate team drafts with cost-based player selection
- **Local Storage Persistence** — All brackets and draft history are saved in your browser
- **Draft Pick Simulator** — Standalone draft simulator for testing team compositions
